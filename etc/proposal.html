To Cook or Not to Cook

2. Description:

This application allows for hungry and indecisive users to find a dish to make/cook for themselves or to find restaurants near them that make the dish instead.

3. Utilized Web services:

  • Google Places
  • Yummly
  • Recipe Puppy

4. Full list of Features:

  • types of food/cuisine drop down box
  • allergens options check box
  • search bar and button
  • image of dish
  • google map of nearby restaurants
  • search auto complete
  • search reset button
  • search voice recognition
  • next/previous page buttons
  • actual links to where the dish came from
  • filter restaurants by price range, distance range, and by rating.
